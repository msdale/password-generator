## Website

## URL image
![Password-Generator](./assets/images/password-generator.png)

### Instructions
#### (1) Press Generate-Password Button
#### (2) Enter password length (enter a number and hit OK)

![Choose password length](./assets/images/choose-password-length.png)

#### (3) Confirm password length (hit 'OK')

![Choose password length](./assets/images/confirm-password-length.png)

#### (4) Choose character types (any of 'l' 'u' 'n' 's' or 'a' alone (hit OK))

![Choose character types](./assets/images/choose-character-types.png)

#### (5) Confirm all choices (hit OK)

![Confirm all choices](./assets/images/confirm-all-choices.png)

#### (6) Respond to notification of password generation (hit OK)

![Respond to notification](./assets/images/notification-of-generated-password.png)

#### (7) Copy and use your password (copy and paste...or memorize (HA!))

![Copy and use password](./assets/images/password-is-generated.png)
